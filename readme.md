# 🕷️ NearbyHero

NearbyHero is a community-driven help platform where people can request assistance and nearby volunteers (heroes) can respond and support them.

The goal of this project is to strengthen neighborhoods by connecting individuals who need help with trusted community members who are willing to assist.

---

## 🚀 Project Overview

NearbyHero allows users to:

- 📝 Create help requests (grocery pickup, tech support, transportation, etc.)
- 🔍 Browse nearby help requests
- 🤝 Offer help as a community hero
- 💬 Communicate securely
- ⭐ Build reputation through ratings and feedback

This platform promotes trust, safety, and collaboration within local communities.

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios (API communication)

### Backend
- FastAPI
- MongoDB Atlas
- Motor (Async MongoDB Driver)
- JWT Authentication
- Passlib (Password Hashing with Bcrypt)

## 🔐 Authentication System

- User Registration
